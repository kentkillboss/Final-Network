## Cài đặt

- `Bước 1`: Clone project
- `Bước 2`: Mở terminal chạy `npm install`
- `Bước 3`: Tiếp tục chạy theo từng lệnh `cd client` + `npm install` + `cd ..`
- `Bước 4`: Chạy chương trình với `npm run dev`
